ℹ️ 关于

AI 伴侣 v2.0

基于 Mini-Agent 框架的端到端 AI 伴侣系统，支持活人感八维度、HMM 状态机、MBTI 人格特化、关系阶段推进等核心能力。

版权声明

本项目为个人研究用途，不构成任何形式的产品或服务承诺。所有 AI 生成内容仅供实验参考。

引用的开源项目

  • Mini-Agent — 轻量级异步 Agent 框架（521 行）
  • FastAPI — Python 异步 Web 框架
  • lark-oapi — 飞书开放平台 Python SDK
  • httpx — 异步 HTTP 客户端
  • pytest — Python 测试框架

技术栈

Python 3.11 · FastAPI · WebSocket · asyncio · Mini-Agent · 飞书长连接
